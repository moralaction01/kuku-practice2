# フラッシュ九九アプリ

![Kuku Practice App](https://img.shields.io/badge/License-MIT-blue.svg)
![React](https://img.shields.io/badge/React-19-blue.svg)
![Tailwind CSS](https://img.shields.io/badge/Tailwind%20CSS-4-blue.svg)

児童向けの楽しく学べる九九練習アプリです。1の段から9の段までを、自動フラッシュモードまたは手動練習モードで練習できます。

## 特徴

- **段別練習**：1〜9の段を個別に選んで練習
- **複数の表示パターン**
  - あがり九九（1×1 から 1×9）
  - さがり九九（9×9 から 9×1）
  - ばらばら九九（ランダム順）
- **2つの学習モード**
  - **手動練習モード**：「答えを見る」「次へ」ボタンで自分のペースで進行
  - **自動フラッシュモード**：一定間隔で問題と答えが自動で切り替わる
- **速度調整**：0.5秒、1秒、2秒、3秒から選択可能（自動モード時）
- **音声フィードバック**：問題表示・答え表示時にビープ音（オン/オフ切り替え可能）
- **進捗表示**：現在の問題番号と進捗バー
- **レスポンシブデザイン**：スマートフォン・タブレット・パソコンで利用可能
- **キッズフレンドリー**：明るく親しみやすいデザイン

## 使い方

### 基本的な操作

1. **段を選ぶ**：左側の「段を選ぶ」から1〜9の段を選択
2. **モードを選択**：「練習枠（手動）」または「自動フラッシュ」を選択
3. **表示パターンを選択**：「あがり九九」「さがり九九」「ばらばら九九」から選択
4. **練習開始**：
   - 手動モード：「答えを見る」ボタンで答えを表示、「次へ」で次の問題へ
   - 自動モード：「開始」ボタンで自動再生開始

### 手動練習モード

自分のペースで九九を練習できます。

1. 問題が表示されます
2. 「答えを見る」ボタンをクリックして答えを確認
3. 「次へ」ボタンで次の問題に進みます
4. すべての問題が終わるとリセットされます

### 自動フラッシュモード

一定間隔で問題と答えが自動で切り替わります。

1. 速度を選択（0.5秒、1秒、2秒、3秒）
2. 「開始」ボタンをクリック
3. 自動で問題→答え→次の問題と進みます
4. 「一時停止」ボタンで一時停止、「リセット」で最初に戻ります

### 音声フィードバック

ヘッダーの「音あり/音なし」ボタンで音声フィードバックのオン/オフを切り替えられます。

## インストール

### 必要な環境

- Node.js 18以上
- npm または pnpm

### セットアップ

```bash
# リポジトリをクローン
git clone https://github.com/yourusername/kuku-practice-app.git
cd kuku-practice-app

# 依存関係をインストール
pnpm install

# 開発サーバーを起動
pnpm dev
```

ブラウザで `http://localhost:3000` にアクセスしてください。

## ビルド

本番用にビルドする場合：

```bash
pnpm build
```

`dist` ディレクトリに本番用ファイルが生成されます。

## デプロイ

### GitHub Pages へのデプロイ

このアプリは静的ファイルのみで構成されているため、GitHub Pages でホストできます。

1. リポジトリの Settings → Pages を開く
2. Source を「GitHub Actions」に設定
3. 自動的にビルドとデプロイが実行されます

### その他のホスティング

Vercel、Netlify、その他の静的ホスティングサービスでもデプロイ可能です。
